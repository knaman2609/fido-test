# Todo App

A simple, clean Todo List application built with TypeScript.

## Quick Start

```bash
# Install TypeScript (if not already installed)
npm install

# Build the project
npm run build

# Serve the app
npm start
```

Then open http://localhost:3000 in your browser.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- TypeScript compiler (`tsc`)

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

## Running the App

### Option 1: Using npm scripts (Recommended)

```bash
# Build and serve in one command
npm start
```

### Option 2: Manual steps

```bash
# Compile TypeScript to JavaScript
tsc

# Serve the app (using any static server)
npx serve . -p 3000
# OR
python -m http.server 3000
# OR simply open index.html in your browser
```

### Option 3: Development mode (auto-rebuild on changes)

```bash
# Terminal 1: Watch for TypeScript changes
npm run watch

# Terminal 2: Serve the app
npm run serve
```

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm run build` | Compile TypeScript files to `dist/` |
| `npm run watch` | Compile in watch mode (auto-rebuild on changes) |
| `npm run serve` | Serve the app on http://localhost:3000 |
| `npm start` | Build and serve the app |
| `npm run clean` | Remove the `dist/` folder |

## Project Structure
