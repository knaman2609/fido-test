# TypeScript Todo App

A simple, interactive Todo application built with TypeScript and vanilla JavaScript. Manage your tasks with an easy-to-use web interface.

## Features

- Add new todos
- Mark todos as complete/incomplete
- Delete todos
- Filter todos by status (All, Active, Completed)
- Persistent storage using localStorage
- Clean, responsive UI

## Quick Start

```bash
# Install dependencies
npm install

# Build and run
npm start
```

Then open your browser to `http://localhost:3000`

## Prerequisites

- [Node.js](https://nodejs.org/) (version 14 or higher)
- npm (comes with Node.js)

## Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Install dependencies:

```bash
npm install
```

## Running the App

### Option 1: Build and Serve (Recommended)

```bash
npm start
```

This compiles TypeScript and starts a local server at `http://localhost:3000`.

### Option 2: Build Only

```bash
npm run build
```

Then open `index.html` directly in your browser.

### Option 3: Development Mode (Watch)

```bash
npm run watch
```

In another terminal:
```bash
npm run serve
```

The `watch` command automatically recompiles TypeScript when you make changes.

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm run build` | Compile TypeScript to JavaScript |
| `npm run watch` | Compile in watch mode (auto-rebuild on changes) |
| `npm run serve` | Start a local development server on port 3000 |
| `npm start` | Build and serve the application |
| `npm run clean` | Remove the compiled `dist/` directory |


## How It Works

1. **TypeScript Compilation**: Source files in `src/` are compiled to JavaScript in `dist/` using the TypeScript compiler (`tsc`).
2. **Module System**: Uses ES modules (type: "module") for clean imports/exports.
3. **Local Storage**: Todos are persisted in the browser's localStorage.
4. **Event-Driven UI**: User interactions trigger updates to the todo list and UI re-renders.

## Development

To make changes to the application:

1. Edit files in the `src/` directory
2. Run `npm run watch` to auto-compile on changes
3. Refresh your browser to see updates

## Troubleshooting

### "Cannot find module" errors
Make sure you've run `npm install` to install TypeScript.

### "dist/ folder not found"
Run `npm run build` to compile TypeScript files.

### Changes not reflecting
- Ensure you're running `npm run watch` for auto-compilation
- Hard refresh your browser (Ctrl+F5 or Cmd+Shift+R)
- Check browser console for errors
